# Maze Runner X - Setup Instructions

## Prerequisites

- Python 3.7 or higher (tested with Python 3.13.7)
- pip (Python package manager)

## Installation

### 2. Create a Virtual Environment (Recommended)

**On Linux/macOS:**
```bash
python3 -m venv venv
source venv/bin/activate
```

**On Windows:**
```cmd
python -m venv venv
venv\Scripts\activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

This will install pygame (the only external dependency required).

## Running the Game

**On Linux/macOS:**
```bash
python3 menu.py
```

**On Windows:**
```cmd
python menu.py
```
